CI/CD là gì?

Tăng tốc quy trình phát triển với Continuous Integration & Delivery

CI (Continuous Integration) là phương pháp tự động build và test code mỗi khi developer push thay đổi lên repository. Điều này giúp phát hiện lỗi sớm và đảm bảo code luôn ở trạng thái hoạt động.

CD (Continuous Delivery/Deployment) là bước tiếp theo, tự động đưa code đã được test thành công lên môi trường staging hoặc production.

Xem Pipeline Công nghệ sử dụng
.github/workflows/deploy.yml 
name: Deploy to ECS
on:
  push:
    branches: [main]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Build Docker Image
        run: docker build -t app .
      - name: Push to ECR
        run: aws ecr push
      - name: Deploy to ECS
        run: aws ecs update-service
Quy trình CI/CD

Các bước trong Pipeline

Mỗi lần push code, pipeline sẽ tự động thực hiện các bước sau để đưa ứng dụng lên production.

1. Source Code

Developer push code lên GitHub repository, kích hoạt GitHub Actions workflow tự động.

2. Build & Test

Tự động chạy unit test, integration test và kiểm tra chất lượng code với các công cụ linting.

3. Docker Build

Build Docker image từ Dockerfile, tối ưu multi-stage build để giảm kích thước image.

4. Push to ECR

Đẩy Docker image lên Amazon ECR — registry lưu trữ container image an toàn và có khả năng mở rộng.

5. Deploy ECS

Cập nhật ECS Service với image mới, thực hiện rolling update để đảm bảo zero downtime.

6. Health Check

Kiểm tra sức khỏe ứng dụng sau deploy, tự động rollback nếu phát hiện lỗi.

Công nghệ sử dụng

Bộ công cụ DevOps hiện đại

Kết hợp các công cụ mạnh mẽ nhất để xây dựng pipeline CI/CD hoàn chỉnh và đáng tin cậy.

GitHub Actions

Nền tảng CI/CD tích hợp sẵn trong GitHub, dễ cấu hình với YAML workflow files.

Docker

Container hóa ứng dụng, đảm bảo chạy nhất quán trên mọi môi trường từ dev đến production.

Amazon ECR

Container Registry của AWS, lưu trữ Docker image an toàn, tích hợp chặt chẽ với ECS.

Amazon ECS

Dịch vụ quản lý container của AWS, hỗ trợ auto scaling và load balancing tự động.
